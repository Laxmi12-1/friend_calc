# ❤️ Friendship Calculator

## Overview

Friendship Calculator is a full-stack web application built using Python Flask, HTML, CSS, and SQLite. The application calculates friendship compatibility between two people based on their names and displays the result through an attractive and responsive user interface.

The project features a modern glassmorphism design, animated floating hearts, a circular friendship meter, and a database to store calculation history.

---

## Features

- ❤️ Friendship compatibility calculation
- 🎨 Modern glassmorphism user interface
- 💜 Animated floating hearts
- 📊 Circular friendship percentage meter
- 📱 Mobile-responsive design
- 🗄️ SQLite database integration
- 📜 Recent calculation history
- ⚡ Built with Python Flask

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Friendship-Calculator.git
```

2. Navigate to the project directory

```bash
cd Friendship-Calculator
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open in browser

```text
http://127.0.0.1:5000
```
